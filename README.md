# Smart Expense Manager 💰

## 🚀 Live Demo

https://smart-expense-manager-seven.vercel.app/

## 🧠 Problem Statement

Managing personal expenses is often manual and lacks insights. This app helps users track, analyze, and optimize their spending using AI.

## ✨ Features

* 🔐 JWT Authentication
* 💰 Expense CRUD
* 📊 Dashboard with Charts
* 🧠 Smart Insights
* 🤖 AI Auto Categorization (Gemini)
* 🌗 Dark/Light Mode
* 📱 Fully Responsive UI

## 🛠 Tech Stack

* Next.js 16 (App Router)
* TypeScript
* MongoDB (Mongoose)
* Tailwind CSS
* React Query
* Gemini AI

## 🧩 Architecture

* API Routes (Backend)
* React Components (Frontend)
* JWT Auth Middleware
* Modular structure

## 🔐 Security

* Password hashing (bcrypt)
* JWT authentication
* Input validation (Zod)

## 🚀 Future Improvements

* Budget alerts
* Multi-user collaboration
* Export reports (PDF/CSV)
